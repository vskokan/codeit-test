# codeit-task
## Description
Registration / Login / Logout forms <br>

### Frontend
**Vue.js** <br>
No additional libraries

### Backend
**Node.js (v10.19.0)**, **Express**, **mysql2** and **bcrypt** (for hashing passwords) <br>
**MySQL v8.0.26**

## Setup

1. Go to project folder

> cd codeit-task

2. Install Dependencies

> npm run setup

3. Create .env with following content to configurate port

````
PORT=3000
````

## Run

1. Start Server

> npm start

2. Open in url or using an Endpoint tester (postman)

> http://127.0.0.1:3000/
